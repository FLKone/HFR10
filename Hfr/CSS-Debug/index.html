Aucune beta de 6/8/2013 jusqu'à la release 0.1

Citation :


Le développement de la branche privée a été stoppé à partir de la Beta 2 pour une raison simple: j'ai décidé de rétroporter l'app pour Windows 8.0 et non Windows 8.1. Seulement voilà, l'environnement de dev de 8.0 est tellement anémique et limitatif que la moitié du code fonctionnant dans WP7 (avec HFR 7) et W8.1 ne tournait pas sous 8.0 en raison du manque important d'API.
J'ai donc du changer complètement le code métier d'HFR, avec un autre concept. Au lieu d'afficher les topics dans un WebBrowser/WebView, c'est à dire en parsant le HTML pour récréer de l'HTML, j'ai décidé de convertir le HTML en éléments natifs du Framework.
Si vous avez un

test/>, son contenu sera utilisée dans une TextBlock créée sur le moment et qui verra ses propriétés Text et Style modifiées.
Sur le papier, ça paraît simple, en réalité, il y a eu un travail de fond assez conséquent. J'ai passé une nuit dans les locaux de l'entreprise So@t (très sympas par ailleurs) où j'ai pu me faire aider par des plus compétents que moi. A 8 heures le lendemain, une version alpha du moteur était prête
Y'en a pour un peu plus d'un millier de lignes de code, c'est pas optimisé et le rendu est moche, mais il fonctionne et sera vite amélioré, rassurez vous.
